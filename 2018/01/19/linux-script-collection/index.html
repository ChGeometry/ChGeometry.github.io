LAMP一键安装包

LAMP 一键安装包是一个用 Linux Shell 编写的可以为 CentOS/Debian/Ubuntu 系统的 VPS 或服务器安装 LAMP(Linux + Apache + MySQL/MariaDB/Percona + PHP) 生产环境的 Shell 程序

OneinStack

搭建网站环境用它就够了，一键安装Apache、Nginx、Mysql、Tomcat…等，比如流行的LNMP环境。注意：该脚本不支持WEB面板，若对Linux不是很熟悉的朋友建议使用宝塔/AMH等替代。
